## 🛠 技術棧 (Tech Stack)

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Deployment:** GitHub Pages
- **Engine:** Client-side parsing logic

## 🌟 功能特點 (Features)

- [ ] 介面簡潔，無廣告干擾。
- [ ] 純前端解析，保障隱私。
- [ ] 支援多平台圖像與影像內容檢索。

## 🚀 如何開始 (How to use)

1. 複製目標平台的連結。
2. 貼上至 DownloaderX 搜尋框。
3. 點擊解析並選擇所需的畫質下載。
